# วาว Print Center — ระบบรายรับ-รายจ่าย

เว็บแอป (PWA) สำหรับบันทึกรายรับ-รายจ่ายร้าน พร้อมสรุปยอดและซิงค์ข้อมูลข้ามเครื่องผ่าน Google Sheet

---

## โครงสร้างไฟล์

```
waow-print-center/
├── index.html        หน้าเว็บหลัก
├── styles.css         ดีไซน์/ธีม
├── app.js             โค้ดทำงานหลัก (เชื่อม API, บันทึก, สรุป, export)
├── config.js           ที่ตั้งค่า API_URL ← ต้องแก้ก่อนใช้งาน
├── manifest.json       ตั้งค่า PWA (ติดตั้งเป็นแอปได้)
├── sw.js                Service Worker (ใช้งาน offline ได้บางส่วน)
├── Code.gs              ตัวจัดการ backend ที่ต้องนำไปวางใน Google Apps Script
└── icons/                ไอคอนแอปทุกขนาด
```

---

## ขั้นตอนติดตั้ง (ทำตามลำดับ)

### 1) สร้าง Google Sheet + Apps Script (backend)

1. ไปที่ [sheets.google.com](https://sheets.google.com) สร้างชีตใหม่ ตั้งชื่อ เช่น **"วาว Print Center - บัญชี"**
2. เมนู **ส่วนขยาย (Extensions) → Apps Script**
3. ลบโค้ดเริ่มต้นในไฟล์ `Code.gs` ที่เปิดมาให้หมด แล้ว**คัดลอกเนื้อหาทั้งหมดจากไฟล์ `Code.gs`** ที่แนบมาในโปรเจกต์นี้ วางแทน
4. กดรันฟังก์ชัน `setup` หนึ่งครั้ง (เลือกฟังก์ชันจาก dropdown ด้านบน แล้วกด ▶ Run)
   - จะมีหน้าต่างขอสิทธิ์ → กด **Review permissions** → เลือกบัญชี Google → กด **Allow / อนุญาต**
   - หลังรันเสร็จ กลับไปที่ Google Sheet จะเห็นชีตย่อย 4 อัน: `รายรับ`, `รายจ่าย`, `รายการ_รายรับ`, `รายการ_รายจ่าย`
5. กด **Deploy → New deployment**
   - Select type: **Web app**
   - Execute as: **Me**
   - Who has access: **Anyone**
   - กด **Deploy**
6. คัดลอก **Web app URL** ที่ได้ (ลงท้ายด้วย `/exec`) เก็บไว้ใช้ในขั้นตอนถัดไป

> หากแก้ไข `Code.gs` ภายหลัง ต้องกด **Deploy → Manage deployments → แก้ไข (ดินสอ) → Version: New version → Deploy** ใหม่ทุกครั้ง ไม่เช่นนั้นแอปจะยังเรียกใช้โค้ดเวอร์ชันเก่า

### 2) ตั้งค่า config.js

เปิดไฟล์ `config.js` แล้วแก้บรรทัด `API_URL` เป็น URL ที่คัดลอกมาจากขั้นตอนที่ 1:

```js
const CONFIG = {
  API_URL: 'https://script.google.com/macros/s/xxxxxxxxxxxxxxxx/exec',
  STORE_NAME: 'วาว Print Center',
};
```

### 3) อัปโหลดขึ้น GitHub แล้วเปิดผ่าน GitHub Pages

1. สร้าง repository ใหม่บน GitHub (public หรือ private ก็ได้)
2. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo (รวมโฟลเดอร์ `icons/`)
3. ไปที่ **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: `main` / โฟลเดอร์ `/ (root)`
   - กด **Save**
4. รอสักครู่ จะได้ลิงก์ เช่น `https://ชื่อบัญชี.github.io/ชื่อ-repo/`
5. เปิดลิงก์นั้นบนมือถือหรือคอม — ถ้า config.js ตั้งค่าถูกต้อง แอปจะดึงรายการและข้อมูลจาก Google Sheet ได้ทันที

### 4) ติดตั้งเป็นแอป (PWA)

- **มือถือ Android (Chrome):** เปิดลิงก์ → เมนู ⋮ → "ติดตั้งแอป" / "เพิ่มไปยังหน้าจอหลัก"
- **iPhone (Safari):** เปิดลิงก์ → ปุ่มแชร์ 🔼 → "เพิ่มไปยังหน้าจอโฮม"
- **คอมพิวเตอร์ (Chrome/Edge):** เปิดลิงก์ → ไอคอนติดตั้ง ⊕ ที่แถบ URL ด้านขวา

หลังติดตั้งแล้ว แอปจะมีไอคอน "วาว Print Center" และเปิดแบบเต็มจอเหมือนแอปทั่วไป ทุกเครื่องที่ติดตั้งจะดึง/บันทึกข้อมูลจาก Google Sheet เดียวกัน จึงซิงค์ถึงกันโดยอัตโนมัติทุกครั้งที่เปิดแอปหรือกดปุ่มรีเฟรช

## วิธีใช้งาน

- **หน้ารายรับ:** เลือกรายการจากดรอปดาวน์ (ถ่ายเอกสาร / ปริ้นงาน / เคลือบเอกสาร / ปริ้นรูป หรือรายการที่เพิ่มเอง) กรอกจำนวนแผ่นและราคา เพิ่มได้หลายแถวในใบเดียว กด **บันทึกรายการ** ระบบจะบันทึกวันที่/เวลา (ชม.นาที) ให้อัตโนมัติ
- **หน้ารายจ่าย:** เหมือนหน้ารายรับ แต่มีแค่ช่องราคา ไม่มีจำนวนแผ่น
- **เพิ่มตัวเลือกรายการใหม่:** กด "+ จัดการตัวเลือกรายการ" ด้านล่างใบเสร็จ พิมพ์ชื่อรายการแล้วกดเพิ่ม รายการนี้จะอยู่ถาวรในดรอปดาวน์ทุกเครื่อง
- **หน้าสรุป:** เลือกช่วงเวลา (รายวัน / รายเดือน / ช่วงวันที่) กด "แสดงสรุป" จะเห็นยอดรายรับ รายจ่าย กำไร/ขาดทุน และรายละเอียดแยกตามรายการ กดปุ่ม **Export Excel** เพื่อดาวน์โหลดไฟล์ .xlsx ของช่วงที่เลือก

## หมายเหตุทางเทคนิค

- ข้อมูลทั้งหมดเก็บใน Google Sheet เป็นแหล่งเดียว (single source of truth) ทุกเครื่องอ่าน/เขียนชีตเดียวกัน จึงเห็นข้อมูลตรงกันเสมอ
- Service Worker (`sw.js`) cache เฉพาะหน้าตาแอป (HTML/CSS/JS/ไอคอน) เพื่อให้เปิดแอปได้แม้เน็ตช้า แต่การบันทึก/อ่านข้อมูลยังต้องมีอินเทอร์เน็ตเสมอ
- หากเปลี่ยนเครื่องหรือเบราว์เซอร์ใหม่ ไม่ต้อง setup อะไรเพิ่ม แค่เปิดลิงก์เดิมแล้วติดตั้งใหม่ ข้อมูลจะอยู่ครบเพราะเก็บบน Google Sheet ไม่ใช่ในเครื่อง
